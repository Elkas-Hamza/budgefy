# Conception Diagrammes

# 📌 Description des Diagrammes UML

Ce projet contient trois diagrammes principaux :

## 1️⃣ Diagramme de Cas d’Utilisation
Il montre les fonctionnalités du système et les interactions entre les acteurs (Utilisateur, Administrateur) et l’application.  Il décrit ce que le système permet de faire.

<img width="895" height="1101" alt="image" src="https://github.com/user-attachments/assets/3bc3855d-4e76-45d7-be51-eccfab2fdeef" />

## 2️⃣ Diagramme de Classes
Il représente la structure du système :  
les classes (Utilisateur, Catégorie, Dépense, Revenu, Tableau de Bord),   leurs attributs, leurs méthodes et les relations entre elles.

<img width="755" height="1202" alt="Diagramme sans nom drawio" src="https://github.com/user-attachments/assets/d90fa9eb-a0c1-4a03-b1dc-822c13b5751b" />

## 3️⃣ Diagramme de Séquence
Il illustre le déroulement des actions dans le temps.  
Il montre comment les objets communiquent entre eux pour réaliser une fonctionnalité (ex: ajouter une dépense).
